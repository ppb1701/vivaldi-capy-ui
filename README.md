# Vivaldi Capy UI Theme

**Version 2.0.0**

Electric Blue dark theme for Vivaldi browser, inspired by the Mastodon Capy UI and Kagi Capy UI themes Features a subtle radial gradient background and the adorable Capy mascot.

## Colors

- **Background**: `#1c1d21` (Tricorn Black)
- **Accent Background**: `#1c2e3d` (Dark Blue-Grey)
- **Foreground/Text**: `#1e80c7` (Electric Blue)
- **Highlight**: `#5f626d` (Medium Grey)
- **Window**: `#255372` (Steel Blue)

## Installation

### From Vivaldi Theme Store (Recommended) 

1. Visit the theme page: [Vivaldi Themes](https://themes.vivaldi.net/themes/VmjvVq1YlYg)
2. Click **Download Theme**
3. Theme will be automatically installed and activated

### Manual Installation

1. Download the `.zip` file from [GitHub Releases](https://github.com/ppb1701/vivaldi-capy-ui/releases/tag/vivaldi-theme)
2. Open Vivaldi Settings (`Ctrl+F12` or `Cmd+,`)
3. Go to **Themes** section
4. Click **Import Theme...**
5. Select the downloaded `.zip` file
6. Theme will be imported with the gradient background and Capy mascot

### Customization

To modify colors or other settings, download the individual files from this repo and edit `settings.json` before importing.

## Features

- Dark theme based on Tricorn Black (`#1c1d21`)
- Electric Blue highlights and accents (`#1e80c7`)
- Radial gradient background with blue glow effect
- Capy mascot logo in as the Home Icon
- 14px border radius for rounded UI elements
- Steel blue window chrome for subtle color throughout

## Files

- `settings.json` - Theme configuration for Vivaldi (importable)
- `background.svg` - Radial gradient background with Capy mascot
- `Home.svg` - 28x28 Capy icon for Home button
- `README.md` - This file
